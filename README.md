# Heart Disease Prediction using Logistic Regression

This project develops a **Heart Disease Prediction** model using **Logistic Regression** to classify whether a patient has heart disease based on various medical attributes. The project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, and performance evaluation.

## Project Description

The objective of this project is to predict the **presence or absence of heart disease** using patient medical data. Logistic Regression is used as the classification algorithm, and the model is evaluated using Accuracy, Precision, Recall, F1-score, and a Confusion Matrix.

## Dataset Description

- **Dataset Name:** Heart Disease Prediction Dataset
- **Source:** Kaggle
- **Total Records:** 270 patients
- **Total Features:** 13 input features and 1 target variable
- **Problem Type:** Binary Classification
- **Target Variable:** Heart Disease (Presence or Absence)
- **Missing Values:** No missing values
- **Objective:** Predict whether a patient has heart disease based on clinical attributes.
- **Application:** Used for early heart disease prediction and healthcare decision support.
